# EduCourse-Portal
EduCourse Portal demonstrates strong backend development skills using Java Spring Boot and MongoDB with a real‑world use case.
EduCourse Portal is a Course and Fee Tracking Web Application designed to manage academic courses and fee details efficiently. The system provides a structured and centralized platform for institutes to handle course management and student-related operations through RESTful APIs.

This project is primarily backend-focused and is built by considering real-world educational institute requirements.

**Key Features**:
  Course Management (Add, Update, View Courses).
  Student Enrollment Management.
  Fee Tracking and Status Monitoring.
  REST API–based communication.
  MongoDB-based data storage.
  Layered architecture (Controller, Service, Repository).

**Tech Stack**:
  Backend: Java, Spring Boot.
  API: RESTful Web Services.
  Database: MongoDB.
  Build Tool: Maven.
  API Testing: Postman.
  IDE: IntelliJ IDEA / VS Code.
